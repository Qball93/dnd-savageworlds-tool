# Dice Roller

Vite React frontend + Node.js backend for rolling dice with modifiers. This is specifically for a homebrewed DnD game using Savage worlds

## Tech
- Vite + React
- Node.js (Express)

## Local Setup
### 1) Install
- Frontend
  - cd frontend
  - npm install
- Backend
  - cd backend
  - npm install


### 2) Run
- Backend: `npm run dev` (or `npm start`)
- Frontend: `npm run dev`
